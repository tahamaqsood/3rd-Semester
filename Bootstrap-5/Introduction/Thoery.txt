============
BOOTSTRAP 5:
============

* Bootstrap 5 is a free front-end framework for faster and easier web development.

* Bootstrap includes HTML and CSS based design templates for typography, forms, buttons,
  tables, navigation, modals, image carousels and many other, as well as optional JS plugins.

* Bootstrap also gives you the ability to easily create the responsive design for your web site
  or web applications.

* Responsive web designs is about creating web sites which automatically adjust themselves to
  look good on all devices, from small phones to large desktop screens.

* Note: We can use the CSS media queries for design the responsive web design.

* But a huge drawback of using CSS media queries that is lengthy coding.

* Bootstrap 5 is the newest version of Bootstrap, which is the most popular HTML, CSS
  and javascript framework for creating responsive, mobile-first websites. 

* Bootstrap 5 is completely free to download and use.

* Bootstrap 5 (Released May/05/2021) is the newest version of bootstrap with the new
  components, faster stylesheet and more responsiveness.
